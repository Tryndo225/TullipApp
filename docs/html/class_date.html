The Date class provides functionality for managing and manipulating date values.

◆ date_

unsigned int Date::date_
protected

Encoded date value.

The date is stored as a 32-bit integer with the following structure:

  • Bits 0-5: Day
  • Bits 6-10: Month
  • Bits 11-31: Year
