The Time class provides functionality for managing and manipulating time values.

◆ time_

unsigned int Time::time_
protected

Encoded time value.

The time is stored as a 32-bit integer with the following structure:

  • Bits 0-7: Seconds
  • Bits 8-15: Minutes
  • Bits 16-23: Hours
  • Bits 24-31: Time zone

00000000 00000000 00000000 00000000

  • ______/ ______/ ______/ ______/
  • Zone Hour Mins Secs
  • Zone Bit 1 = +/-, Rest is amount of 15 mins interval difference from UTC-0
